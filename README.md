# Student Registration & Login System (Java)

##  Project Description
This project is a Java console application that allows users to register and log in. It validates user input such as username, password, and phone number to ensure security and correctness.

The system demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and input validation.


##  Features
- User Registration
- Username validation (must contain underscore and be max 5 characters)
- Password complexity check (uppercase, number, special character, minimum 8 characters)
- Phone number validation (+27 format)
- User Login system
- Login status messages


##  Technologies Used
- Java
- NetBeans IDE



##  How to Run the Project

1. Open the project in NetBeans
2. Build the project
3. Run the `Registration` class
